Práctica 2. Estrategias de búsqueda

Objetivo.-

Comprender el funcionamiento de las principales estrategias de búsqueda no informadas e informadas. Para ello, el alumno debe implementar las búsquedas primero en amplitud, primero en profundidad y A* para el problema de encontrar un camino entre dos vértices de un grafo. El camino será el de longitud mínima cuando se emplee la búsqueda A*.
